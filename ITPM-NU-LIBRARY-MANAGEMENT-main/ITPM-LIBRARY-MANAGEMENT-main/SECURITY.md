# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

If you find a security issue:

1. Do not open a public issue.
2. Share details privately with maintainers.
3. Include reproduction steps, impacted areas, and suggested mitigations if possible.

## Response Expectations

- Initial acknowledgment: within 72 hours
- Triage and risk assessment: as soon as practical
- Patch timeline: based on severity and exploitability
