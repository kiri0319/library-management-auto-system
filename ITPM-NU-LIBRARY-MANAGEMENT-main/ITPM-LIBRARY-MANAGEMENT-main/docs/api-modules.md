# API Modules Overview

## Core domain modules

- `auth`: login, registration, password recovery, token validation
- `users`: role-aware user management
- `books`, `authors`, `categories`: catalog management
- `borrows`, `reservations`, `fines`: circulation and penalties

## Operational modules

- `dashboard`, `reports`, `activity-logs`, `settings`
- `notifications` and Socket.io event dispatching

## Newly expanded modules

- `book-health`
- `librarian-productivity`
- `seats`
- `support-chat`
